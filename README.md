# Multi-Tier CI/CD Project

This project demonstrates a multi-tier web application with continuous integration and deployment (CI/CD) using GitHub Actions.

## 📁 Project Structure


---

## 🚀 Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express
- **CI/CD:** GitHub Actions

---

## 🛠️ How to Run Locally

### 1. Clone the Repository
```bash
git clone https://github.com/Nehaa-priya/multi-tier-ci-cd.git
cd multi-tier-ci-cd
cd backend
npm install
node index.js
cd "../front end"
npm install
npm start

4. Scroll down and click **Commit changes**.
